# ioterm-demo
Demonstrations for IOTerm.

1. [Simple demo](./simple)
2. [WebSocket](./websocket)

### Getting Start
1. Install `webpack`, `webpack-cli` and `ioterm`.
```
npm install webpack webpack-cli --save-dev
npm install ioterm
```

2. Do webpack.
```
npx webpack
```

3. Open `dist/index.html` in browser.
